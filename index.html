Function Generator

Led a team of engineers to deliver a custom commercially viable prototype for the generation of various functions. Wrote firmware in C using AVR, custom onboard analogue controls, and a Python front-end application.

Technical specifications:

  • Frequency range: 1Hz to 1MHz
  • Multiple waveform outputs: sine, square, triangle, sawtooth
  • Adjustable amplitude and offset controls
  • Digital display for precise parameter settings
